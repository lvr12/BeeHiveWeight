# Security Policy

## Supported Versions

This table shows which versions of BeeHiveWeight are currently maintained with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | ✅                 |
| 5.0.x   | ❌                 |
| 4.0.x   | ✅                 |
| < 4.0   | ❌                 |

It is strongly recommended to use a supported version for optimal security and to benefit from the latest patches.

## Reporting a Vulnerability

If you discover a security vulnerability in BeeHiveWeight, please follow these steps :

- **Contact:**  
  Send an email with detailed information to :  
  **quentin.kern@outlook.com**
- **Please include:**  
  - A precise description of the vulnerability and its impact.
  - Steps to reproduce the issue, if possible.
  - Any suggestion for a fix or mitigation, if available.

**Response process**  
- You will receive an acknowledgment within 5 working days.
- Updates regarding investigation and remediation will be provided within 10 working days.
- Once the vulnerability is confirmed, we’ll coordinate a fix and may ask for your collaboration.
- Vulnerabilities will be disclosed only after a patch is released, unless you request otherwise.

Thank you for helping keep BeeHiveWeight and its users secure!
